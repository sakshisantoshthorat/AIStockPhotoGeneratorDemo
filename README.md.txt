# AI Stock Photo Generator

AI-powered stock image generator using:

- Spring Boot
- Spring AI
- React JS
- OpenAI / Gemini API

## Features

- Generate AI images from prompts
- Chat AI integration
- Modern UI
- REST API backend

## Tech Stack

Frontend:
- React
- HTML/CSS

Backend:
- Spring Boot
- Spring AI

## Run Project

Backend:

```bash
mvn spring-boot:run
